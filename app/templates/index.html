SODA BOTTLE PREDICTION

Home | Info | Statistics



It has the following functionalities:

  1. A POST request to the path /predict with a sent photograph returns the probability of belonging to each of the possible classes (supporting 1 or more images).
  2. A GET request to the path /info retrieves a list of currently supported classes that the model can currently detect.
  3. By calling /stats, the user will receive information in JSON format about the number of successful predictions for the last 10 days for each class.
